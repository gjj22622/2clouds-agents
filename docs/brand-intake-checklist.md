# Brand Intake Checklist — 甲方品牌導入確認清單

本清單由 Sophia 或 Jacky 在接案後、品牌腦建立前填寫。填寫完成後，匯入平台成為品牌工作台的初始資料，並作為新人品牌腦閱讀的基礎文件。

**填寫規則：**
- 方括號 `[...]` 為待填入的佔位符。
- 有底線標記的欄位為「新人必讀」項目。
- 任何無法確認的欄位填「待確認」並附上預計確認時間。
- 合規相關欄位不可留空；若不確定，填「需升級 Jacky 確認」。

## Part 7：合規邊界（必填，不可留空）

> 新人進入品牌前必讀。如有任何不確定，填「需升級 Jacky 確認」。

**產業合規要求**

- 適用法規 / 規範：`[例：食品標示法 / 廣告不實規範 / 衛福部廣告準則]`
- 需要前置確認才能發布的內容類型：`[列舉，例：含療效字眼的貼文]`
- 無需特別確認即可發布的內容類型：`[列舉，例：品牌故事 / 活動預告]`

**政治與敏感話題邊界**

- 嚴禁觸碰：`[例：統獨議題 / 宗教比較 / 種族或性別偏見]`
- 謹慎處理（需先確認）：`[例：環保議題 / 動物權利相關 / 特定政策評論]`
- 可操作（低風險）：`[例：一般節慶 / 無政治色彩的社會議題]`

**競品提及規則**

- 禁止：`[例：直接指名競品 / 比較聲稱「我們比 XX 好」]`
- 可接受：`[例：市場趨勢說明（不指名）/ 行業整體情況]`
