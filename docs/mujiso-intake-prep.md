## Sophia 初探會議問題清單

> 在 Stage 8 啟動會議或初次接觸木酢寵物達人時，Sophia 依此清單提問，收集填寫 Component 1-7 所需的資訊。**不用一次問完；可分 1-2 次會議蒐集。**

### 品牌定位與語氣（對應 Component 1）

1. 你們希望消費者看到品牌時，第一個聯想到的形容詞是什麼？
2. 有沒有你們「絕對不想要」的語氣或形象？（太正式 / 太輕浮 / 太科技感）
3. 目前粉絲或客戶最常用什麼詞語形容你們的產品？
4. 在社群上，有沒有哪個品牌的語氣風格是你們希望靠近或明確區隔的？

### 目標受眾（對應 Component 1 + 3）

5. 你們的主力購買客群是誰？（年齡、飼養寵物類型、購買動機）
6. 有沒有你們「不想服務」或「不想靠近」的客群？
7. 客戶最常在什麼情境下買你們的產品？（寵物生病 / 換季 / 看到推薦）
8. 客戶對產品最常有哪些疑慮？（成分來源 / 價格 / 效果驗證）

### 產品與競品（對應 Component 2）

9. 目前主要產品線有哪幾條？各自的定位是什麼？
10. 和市面上競品相比，你們最不一樣的地方在哪裡？
11. 有沒有「不能說」的競品名稱？或不能做比較的方式？
12. 有沒有特別成功的過去廣告或內容，可以作為方向參考？

### 頻道與內容規格（對應 Component 5）

13. 目前主力在哪些平台？各平台的發文節奏大概是？
14. 過去在哪些平台的成效最好？原因是什麼？
15. 有沒有特別希望嘗試或避開的內容格式？（Reels / 直播 / UGC）
16. LINE 官方帳號有在使用嗎？如果有，推播頻率與主要用途是什麼？

### 合規邊界（對應 Component 1 taboos + Part 7）

17. 有沒有曾經收到廣告主管機關的警告或要求修改的經驗？
18. 產品說明或廣告文案，哪些詞語是你們內部明確不用的？
19. 如果消費者問「這個可以治療我的狗的皮膚病嗎？」，你們希望怎麼回應？
20. 有沒有特定的合作 KOL 或合作形式是你們想避開的？

### 客戶關係與溝通習慣（對應 Component 7 + brand-intake-checklist）

21. 老闆最在意的成效指標是什麼？（觸及 / 互動 / 轉換 / 品牌形象）
22. 你們習慣怎麼接受報告？（週報 / 月報 / LINE 口頭 / 正式簡報）
23. 未來和双云的主要溝通管道是什麼？誰是窗口？
24. 有沒有過去和行銷公司合作不愉快的原因，我們應該特別注意？

## Component 1：品牌腦（BrandBrain）— Phase 1 必填

> 全部欄位在 Sophia 完成初探後填寫，藝嘉確認 `taboos` 與合規邊界。

```
voice（品牌語氣）：
  [待填：描述整體語氣定調，例如「親切、專業、生活化」]

audience（目標受眾）：
  [待填：描述主要消費者樣貌，例如「有寵物的 30-45 歲家庭主婦 / 寵物主人」]

offer（核心價值）：
  [待填：描述品牌帶給受眾的核心價值，1-2 句]

taboos（禁忌清單）：
  - [待填：第 1 條禁用詞或禁忌話題]
  - [待填：第 2 條，例：禁止宣稱療效的用詞]
  - [待填：第 3 條，例：禁止比較競品]
  - [待填：第 4 條（如有）]
  - ⚠️ 寵物產品廣告須特別注意：禁止宣稱「治療」「預防疾病」等醫療用語（依廣告法規）

channelRules（頻道規則）：
  - FB：[待填：發文頻率、內容類型、CTA 風格]
  - IG：[待填：圖片風格、Hashtag 策略、限動規格]
  - 其他頻道：[待填或標記「不使用」]

escalationRules（升級條件）：
  - [待填：第 1 條，例：涉及寵物健康聲稱需升級]
  - [待填：第 2 條，例：客戶詢問療效或成分認證需升級]
  - [待填：第 3 條（如有）]
```

## 寵物產業合規提醒

以下為導入前藝嘉需確認的合規邊界（一般通則，不含木酢特定內容）：

**絕對禁止（任何寵物產品品牌）：**
- 宣稱「治療」「預防」「消滅」疾病或害蟲等醫療用語
- 未有科學依據的功效聲稱（例：「100% 有效」「臨床證實」）
- 暗示可替代獸醫診療的建議

**需升級確認的邊界：**
- 含「天然」「有機」「植物性」等字眼的定義與依據
- 使用者見證或評論（需確認是否符合廣告規範）
- 與競品的比較性說法

**填寫時注意：**
- `taboos` 欄位必須包含至少 1 條寵物健康宣稱的禁忌
- `escalationRules` 必須包含「涉及產品功效聲稱時升級藝嘉或 Jacky」
