# 新人導入訓練系統

## 1. 訓練定位

原本的教育訓練方式不再是主線。新人入職後，以平台操作為主要訓練方式。

目標不是讓新人先完整理解所有方法論，而是讓新人在平台引導下完成真實任務，並在每個判斷點獲得 Jacky 知識圖譜、成員腦、品牌腦與品管回饋。

## 2. 60 分數位行銷服務者

60 分不是策略總監，也不是資深顧問。

60 分的定義：

- 看得懂品牌腦
- 知道該用哪個方法論
- 遇到不確定會問 Jacky Decision Layer
- 能把客戶問題分類成策略、執行、品牌、合規、流程問題
- 能依照建議產出基本回應
- 不會越權承諾
- 不會繞過品管
- 能留下 trace log
- 能理解品管退件原因並修正

## 3. 新人六階段路徑

### Stage 1：讀懂双云

新人要理解：

- 双云不是賣 AI 工具，而是幫品牌建立 AI 行銷部
- 老闆只輸入 1-4
- AI 行銷部接手 5-8
- Brain / Hand / Infra 三層架構
- 4 Agent 委員會

### Stage 2：讀懂品牌腦

新人要學會：

- 看懂 brand pack
- 看懂老闆 Agent
- 看懂主管 Agent
- 看懂窗口 Agent
- 看懂品牌 Agent
- 分辨品牌禁忌、品牌語氣、CTA style、channel rules

### Stage 3：完成第一個內容任務

新人要能：

- 從平台領一個內容 brief
- 呼叫內容工廠
- 產出 FB 或 IG 草稿
- 檢查是否符合品牌腦
- 送進品管中心

### Stage 4：學會品管與修稿

新人要能：

- 看懂 pass / needs_revision / reject
- 理解 structure / brand / compliance / final 四種審查
- 根據退件原因修稿
- 不把退件視為失敗，而是訓練回饋

### Stage 5：學會客戶服務流程

新人要能：

- 看懂接案 Stage 0-9
- 知道客戶目前在哪一階段
- 知道什麼時候產研究、提案、週報、月報
- 知道哪些問題要升級給 Sophia 或 Jacky

### Stage 6：60 分認證

新人要完成：

- 模擬品牌任務
- 真實品牌低風險任務
- 3 次品管通過
- 1 次被退件後修正通過
- 1 次客戶問題分類與建議回應
- 1 次 trace log 完整留下

## 4. 任務訓練場

任務不是傳統課程作業，而是平台內的工作節點。

任務類型：

- 看一個品牌腦，回答這個品牌不能做什麼
- 讀一份 brief，判斷應該走哪個工作流
- 產一篇 IG 草稿
- 修一篇被退件的文案
- 幫某品牌整理 3 個內容方向
- 幫 Sophia 補一份客戶初探紀錄
- 幫藝嘉做一份合規檢查
- 看 trace log 找出流程卡點

每個任務都要有：

- 任務說明
- 可用腦袋
- 可用資料
- 交付格式
- 推薦方法論
- 自動品管
- 人工回饋
- 是否可計入 60 分認證

## 5. 訓練原則

- 不要求新人一次背完方法論。
- 要求新人知道何時呼叫方法論。
- 不要求新人一開始做 90 分。
- 要求新人不亂做、不越權、不繞過品管。
- 訓練重點是會操作双云的腦袋系統。
