## 核心萃取

### 双云 AI 行銷部

双云不是在賣 AI 工具，而是在幫每個品牌建立可以消除摩擦力的 AI 行銷部。

### 老闆 1-4 / AI 5-8

老闆負責：

1. 找問題
2. 定邊界
3. 設目標
4. 給方向

AI 行銷部負責：

5. 想策略
6. 排執行
7. 自品管
8. 學回饋

### 4 Agent 委員會

每個客戶都有自己的：

- 老闆 Agent
- 主管 Agent
- 窗口 Agent
- 品牌 Agent

### Skill OS

人只對前台 agent 說需求，不手打多段 prompt。Agent 自動分類、路由、調用 skill、回報下一步；只有例外、退件、缺資料、人工批准時才把人拉進來。

### AGENTS 方法論

平台訓練與工作節點應散布：

- 拆
- 建
- 修
- 串
- 管
- 交

### 品管

所有 publishable output 必須經過 reviewer 與 trace logger。
